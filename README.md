# Projet-CPP
Projet 2025-2026 de l'électif C++ à ECM

Sujet :
Le but de ce projet est de fournir un programme qui permette de gérer un réseau de
bibliothèques. Il faut donc pouvoir représenter les différentes bibliothèques et aussi leur contenu, leurs
adhérents, les emprunts, les échanges entre bibliothèques, les achats, les pertes et la mise au
pilon de certains ouvrages
